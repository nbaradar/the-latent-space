**Qwen3-Coder** is [[Alibaba]]s most agentic code model to date. This agentic CLI model is **OPEN SOURCE** so you can run it locally. Alongside the model, they have also open-sourced a command-line tool for agentic coding: [Qwen Code](https://github.com/QwenLM/qwen-code).
